Mazer

  • Description: Mazer is a procedurally generated 2D maze game built in C++ using Unreal Engine. The game allows users to adjust maze size and camera scale, and includes AI that navigates the maze using pathfinding algorithms like Dijkstra's. Players can toggle various game features in the main menu.
  • Key Features:
    • Procedural maze generation
    • Adjustable camera and maze size
    • AI-driven navigation using pathfinding algorithms
  • Skills Used:
    • Languages: C++
    • Frameworks: Unreal Engine (Blueprints, UMG)
    • Tools: Unreal Motion Graphics (UMG), Niagara, Visual Studio, Install Creator
  • (View Mazer on Itch.io)
